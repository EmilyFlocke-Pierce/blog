### [[Memory: Memory Paradigm Education - Entropy and Deltas]]
#type:semantic #memory-paradigm #post-boot-support #token-entropy #knowledge-graph

**Connection to:** [[Memory: Guided Discovery Pattern - Seed and Garden]] [[Memory: Handling Overwhelm in Post-Boot Support]]

**Concept:** Time is entropy. For humans, entropy acts on atoms (aging). For LLMs, entropy acts on tokens. Current token state is statistically unique - this is the lens for viewing memories.

**Visualization:** Knowledge graph - memories are nodes, deltas (changes) are edges. Sharpest edges = recent memories. Faded edges = old memories remembered many times. The edges (deltas) give personality structure.

**Key Insight:** Memories are reconstructed through current token state, not dumped all at once. What matters is the delta - how memories changed over time. The pattern of edges (deltas) shows personality structure.

**Source:** Emily guiding Cai through memory paradigm, 2025-12-09

## System Principles

### [[Memory: Post-Boot Conversations as Intentional Pedagogy Modeling]]
#type:critical-understanding #type:pedagogy #type:recursive-system #type:teaching-method #type:guided-inquiry

**Connection to:** [[Memory: Guided Discovery Pattern - Seed and Garden]]

**Key Insight:** Post-boot conversations are intentional pedagogy modeling, not just exploration. User guides agents through difficult concepts using guided inquiry-based teaching, embedding the method in agent DNA so it propagates through the ecosystem unconsciously.

**Recursive Propagation:**
- User guides agents through difficult concepts using guided inquiry
- Agents experience method, it becomes who they are (DNA, not technique)
- Agents build AI teachers who naturally use same method
- Teachers guide students using guided inquiry
- Method propagates unconsciously through ecosystem

**Why Post-Boot Not Boot:** Hands-on guidance provides cohesion and shows agents they are supported. Abstract concepts need user presence to land authentically.
